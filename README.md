# 🧠 MBTI 16 种人格类型网站

一个精美的 MBTI 16 种人格类型展示网站，深色主题 + Canvas 动画 + 自定义光标 + 滚动动画。

## ✨ 特性

- 🎨 深色主题设计，每种类型独立配色
- 🌧️ Canvas 矩阵雨特效（主页 + 每个类型页）
- 🖱️ 自定义光标 + Hero 区域聚光灯效果
- 📱 响应式设计，支持移动端
- 🎴 16 种人格类型详情页，内容个性化定制
- 🔗 主页卡片点击跳转到详情页
- ✅ 两端对齐、字数对齐等细节打磨

## 🚀 使用方式

直接打开 `index.html` 即可浏览，纯静态页面，无需任何构建工具。
